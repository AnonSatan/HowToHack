Programming Languages

Scripting, Dynamic Languages

Scripting is perhaps the easiest way to start programming. A script runs progressively through the file, executing the commands as it goes. The popular languages are python, perl, php, and bash. Most scripting languages require an interpreter to be installed on host machine. This makes running any dynamic language on windows tricky, as these languages are not installed by default.

OOP, Static Languages

Object Oriented Programming or OOP, is all the rage these days. What you do is define an object, and take actions on that object. This is really quite powerful if done correctly, and an utter disaster if done incorrectly. The popular languages are C++, C, Java, and any of the .NET languages. These languages require the code to be compiled, which means the resulting code can be run "cross platform" as the frameworks are popular or installed by default.
